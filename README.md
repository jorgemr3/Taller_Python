# Taller_Python


Estructura recomendada:

```
Taller_Python/
  # Taller_Python

  Breve descripción del proyecto.

  Contenido clave:

  - **Breve descripción del proyecto:** Ejercicios y ejemplos del curso de Python que muestran manejo de datos con `pandas`, visualización con `matplotlib`, y ejemplos básicos de programación en `src/` y `notebooks/`.
  - **Instrucciones para clonar el repositorio:**

  ```powershell
  git clone <URL-del-repositorio>
  cd Taller_Python
  ```

- **Cómo instalar dependencias:**

  1. Crear y activar un entorno virtual (recomendado):

  ```powershell
  python -m venv .venv
  .\.venv\Scripts\Activate.ps1
  ```

  2. Instalar dependencias:

  ```powershell
  pip install -r requirements.txt
  ```

- **Cómo ejecutar el código:**

- Ejecutar el módulo principal (desde la raíz del proyecto):

  ```powershell
  python -m src.main
  ```

- O ejecutar el script `main.py`:

  ```powershell
  python main.py
  ```

  Créditos al instructor Dr Adhara Cavazos Matsumoto y materiales del taller. Consulte la sección de créditos en la documentación del curso o contacte al organizador del taller para más detalles.
